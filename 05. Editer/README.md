# 文字編輯器

## 簡介

文字編輯器是 Linux 系統中最重要的工具之一，用於建立、編輯和修改文字檔案。在命令列環境中，文字編輯器是進行系統配置、撰寫腳本和處理檔案的主要方式。

## 常見的文字編輯器

### Nano

Nano 是一個簡單易用的文字編輯器，適合初學者使用。

**特點：**
- 操作簡單，鍵盤快捷鍵顯示在畫面底部
- 不需要學習複雜的模式切換
- 適合快速編輯小型檔案
- 預設安裝在大多數 Linux 發行版

**適用情境：**
- 初學者編輯設定檔
- 快速修改小型文字檔案
- 不需要複雜功能的簡單編輯

詳細使用說明請參考：[nano.md](nano.md)

### Vi / Vim

Vi 是 Unix 系統中的標準文字編輯器，Vim（Vi Improved）是 Vi 的增強版本。

**特點：**
- 功能強大，支援高度自訂
- 高效的文字編輯能力
- 支援多種模式（一般模式、插入模式、命令模式）
- 學習曲線較陡，但掌握後效率極高
- 幾乎所有 Linux 系統都預裝

**適用情境：**
- 進階使用者和開發者
- 需要高效編輯大型檔案
- 系統管理任務
- 程式開發

詳細使用說明請參考：[vim.md](vim.md)

### 其他編輯器

#### Emacs

Emacs 是一個功能強大的可擴展編輯器，不僅是編輯器，更像是一個完整的開發環境。

**特點：**
- 功能極其豐富
- 可透過 Lisp 擴展
- 學習曲線陡峭
- 資源消耗較大

#### Gedit / Kate

圖形介面的文字編輯器，適合桌面環境使用。

**特點：**
- 圖形化介面，操作直觀
- 語法高亮顯示
- 適合桌面使用者

## 選擇建議

### 初學者

建議從 **Nano** 開始，因為：
- 操作簡單，容易上手
- 快捷鍵顯示在畫面上
- 不需要學習複雜的概念

### 進階使用者

建議學習 **Vim**，因為：
- 功能強大，效率高
- 幾乎所有系統都有安裝
- 掌握後可以大幅提升編輯效率

### 桌面使用者

可以使用圖形化編輯器如 **Gedit** 或 **Kate**，操作更直觀。

## 基本操作概念

無論使用哪種編輯器，都需要了解：

1. **開啟檔案**：在命令列輸入編輯器名稱和檔案路徑
2. **編輯內容**：輸入或修改文字
3. **儲存檔案**：使用相應的快捷鍵或命令
4. **退出編輯器**：完成編輯後退出

## 快速參考

### Nano 基本操作

```bash
# 開啟檔案
nano filename.txt

# 快捷鍵（顯示在畫面底部）
Ctrl + O  # 儲存
Ctrl + X  # 退出
Ctrl + W  # 搜尋
```

### Vim 基本操作

```bash
# 開啟檔案
vim filename.txt

# 基本命令
i          # 進入插入模式
Esc        # 回到一般模式
:w         # 儲存
:q         # 退出
:wq        # 儲存並退出
```

## 學習路徑

1. **初學階段**：學習 Nano 的基本操作
2. **進階階段**：學習 Vim 的基本模式和使用
3. **精通階段**：掌握 Vim 的高級功能和自訂配置

每個編輯器都有其適用場景，建議根據自己的需求和經驗選擇合適的工具。
